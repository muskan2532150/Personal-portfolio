## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:

- Node.JS
- Git
- Broswer to Display
- IDE to run and edit the code

### Setup

Clone this repository to your desired folder:

```
  git clone  https://github.com/muskan2532150/Personal-portfolio.git
```

change directory to cloned folder

```
  cd fitmeup
```

### Install

Install project dependecies with:

```
  npm install
```

-

### Usage

To run the project, execute the following command:

```
  npm start
```

- After running `npm start`, open your browser and enter this address http://localhost:3000/

- Congratulations! your're running **_Portfolio_**
